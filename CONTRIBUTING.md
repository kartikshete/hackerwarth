# Contributing to Aether Credit

Thank you for your interest in contributing! We want to keep Aether Credit at the cutting edge of AI-driven fintech.

## Code of Conduct
Please be respectful and professional in all interactions.

## How Can I Contribute?
- **Reporting Bugs**: Open an issue with clear steps to reproduce.
- **Suggesting Features**: We love new agent ideas! Open an issue to discuss.
- **Pull Requests**:
    1. Fork the repo.
    2. Create a branch (`feature/amazing-feature`).
    3. Commit your changes with clear messages.
    4. Push to the branch.
    5. Open a Pull Request.

## Coding Standards
- Use Clean Code principles.
- Use `lucide-react` for icons.
- Ensure all components are responsive.
- Document all new AI agents in `docs/architecture.md`.
